# CIS267\_FinalGroupProject





Overview:

&nbsp;	Play as a human protecting his bas by placing towers 

&nbsp;	and grabbing weapons to fight waves of monsters.



Controls: 

&nbsp;	Movement: WASD

&nbsp;	R: Repair Hammer

&nbsp;	F: Health Potion

&nbsp;	E: Interact with buildings

&nbsp;	1-6: Weapon hotkeys

&nbsp;	F1-F4: Tower Hotkeys 

&nbsp;	TAB: Switch Tower Types

&nbsp;	Left click: Attack



Special Items:

&nbsp;	Repair Hammer: Repairs towers

&nbsp;	Health Potion: Heals player.
